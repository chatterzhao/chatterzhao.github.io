如何配置 能同时提交代码到 GitHub 和码云

为什么要同时，对国内平台的一个支持，是觉得自己英语不好，在中文平台找资源学起来快，但是目前平台还不强，很多中国人做的项目都没有推送到码云上。怎么做到呢，需要：

  • 取消git config 用户及邮箱的全局配置
  • 生成多个平台的密钥，各自上传到各自平台
  • 在项目git 下的config 中同时配置remote
